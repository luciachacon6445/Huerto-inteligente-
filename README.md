# Huerto-inteligente

## Descripción 

Aplicación web diseñada para gestionar y dar seguimiento de un huerto por medio de visión de computadora y con el uso de IA obtener un diagnostico rápido de nuestra planta. Permite registrar, monitorear riego y controlar el estado de las plantas

## Caracteristicas 

* Registro de cultivo

* Seguimiento de riego

* Historial de seguimiento
  
* Vizualización de datos

* Información general de la planta a primera vista

* Recomendaciones para una mejor vida
